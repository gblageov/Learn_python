Isolated Problems

Clone the repository. Solve every task in separate file.

Use 2 separate functions in single file for the tasks that have 2 parts.

1) Rectangle Area Calculator

Create a function calculateRectangleArea that accepts 2 arguments:

  1. width
  2. height

Return the area of the rectangle.

2) Circle Area Calculator

Create a function calculateCircleArea that accepts the radius as an argument.

Return the area of the circle base on the formula πr2

3) Is Allowed to Drive

Create a function isAllowedToDrive that determinates whether a person is allowed to drive a car. The function should accept the following arguments:

  1. age (number)
  2. hasDrivingLicense (boolean)
  3. hasConsumedAlcohol (boolean)

Return true when the person is at least 18 years old, has a driving license and has not consumed alcohol.

4) Even Numbers Printer

Createa a function that prints all even numbers between 2 and 101 in the console.

5) Paint Calculator

Part 1
Create a function that calculates the amount of paint required for painting a rectangular ceiling of a room. One m2 requires 0.66 liters of paint. The function should accept 2 arguments(width and length) and return liters of paint required for painting. Example: 4, 3 => 7,92

Part 2
Add third argument height and return the amount of paint needed for painting all walls of the room, and the ceiling(without the floor). Example: 4, 3, 5 => 54,12

6) Currency Converter

Part 1
Create a function that converts amount of Bulgarian levs (BGN) into euro (EUR). The exchange rate is 1.9558.

Part 2
Add second argument that allows changing of the result currency to one of the following:

  • USD (exchange rate 1.76)
  • GBP (exchange rate 2.03)
  • CHF (exchange rate 1.69)

Represent the currencies(along with the exchange rates) with a javascript object.

Make sure that other currencies do not result in errors.

7) Taxis Count Calculator

Part 1
Create a function that calculates the number of taxis needed for transporting given amount of people. Single taxi can transport up to 3 people.

Example input and output:

  • 1 => 1
  • 3 => 1
  • 4 => 2
  • 10 => 4

Part 2
Extend the solution in part 1 so the types of taxis you can call are either "big" or "small". The big taxis can transport up to 6 people, and the smaller ones -- up to 3 people.

The new function should return an array where the first element is the number of smaller taxis, and the second element - the number of bigger taxis.

Example input and output:

  • 1 => [1, 0]
  • 3 => [1, 0]
  • 4 => [0, 1]
  • 10 => [0, 2]

8) Interest Calculator

Part 1
Create a function that calculates the interest of a deposit. The interest rate is fixed to 4.3 percent per year.

Example input and output:

  • 100 => 104,3
  • 0 => 0
  • 1000 => 1043

Part 2
Add second argument that allows caller to specify number of years for the deposit. You should calculate compound interest, so the interest from the first year should be included in the initial amount for the second year(i.e. the interest has interest).

Do not use the formula for that; instead, create a loop and calculate the amount for each year until you get to the last year.

Example input and output:

  • 100 => 108,78
  • 0 => 0
  • 1000 => 1087,8

9) Password Strength Indicator

Create a function that returns the strength score of a password. The score should be calculated in the following way:

  • for each letter: add 1 point
  • for each digit: add 2 points
  • for everything else: add 3 points
  • if the password has a letter, a number, and a special character: multiply the result by 2

Example input and output:

  • TEST => 4
  • TesT => 4
  • test => 4
  • test6 => 6
  • test@ => 7
  • test6@ => 18

Use the following function to check whether a particular character is a letter: 

/**
 * Return true when the passed argument is single word character.
 */ 
function isLetter(character) {
    return /^[a-z]{1}$/i.test(character);
}

10) Multiplication table

Create a function that return a 2-dimensional array containing the multiplication table.

Hint: use console.table() to review your results.
